# 项目简介

## 核心功能
- 用户登录注册
- 商品分类展示
- 商品详情浏览
- 加入购物车
- 购买商品
- 商品结算
- 商品信息管理
- 用户信息管理
- 订单管理
- 购物车管理
- 轮播图管理

## 适用场景
适用于需要搭建一个简单的二手交易平台，实现用户之间的商品交易。

## 技术特点
- 使用 Django 框架构建
- 基于 MySQL 数据库存储
- 采用 HTML, CSS, JS 实现前端界面展示
